Python dependencies
===================
- **Platform:** macOS and Linux
- **Python:**  3.7+

Install package
===============
To install the ``crypto_email`` package:

.. code-block:: bash

   pip install git+https://github.com/raul23/crypto-email#egg=crypto-email

Uninstall package
=================
To uninstall only the ``crypto_email`` package:

.. code-block:: bash
 
   crypto_email -u
