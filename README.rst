.. 
  Python dependencies
  ===================
  - **Platform:** macOS and Linux
  - **Python:**  3.7+

    Install package
    ===============
    To install the ``cryptoemail`` package:

    .. code-block:: bash

       pip install git+https://github.com/raul23/crypto-email#egg=crypto-email

  Uninstall package
  =================
  To uninstall only the ``cryptoemail`` package:

  .. code-block:: bash

     cryptoemail -u
