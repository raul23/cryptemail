Install package
===============
To install ``monitoring`` package:

.. code-block:: bash

   pip install git+https://github.com/raul23/mac-monitoring#egg=mac-monitoring

Uninstall package
=================
To uninstall ``monitoring`` package:

.. code-block:: bash

   pip uninstall mac-monitoring
